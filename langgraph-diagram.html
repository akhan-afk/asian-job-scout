get_regions()
→ str
Returns all available regions and the date of their most recent summaries. Queries MongoDB summaries collection per region.
called when user asks what regions/countries are available
list_jobs()
region: str → str
Returns all latest job summaries for a region. Queries most recent summarized_date in MongoDB. Falls back to data/<region>/summaries_*.txt.
called when user asks to see jobs in a country
search_jobs()
query: str, region?: str → str
Keyword search across summaries body + tags[] + stack[]. Searches all regions if region omitted. MongoDB-first, txt fallback.
called when user searches for a role or skill
trigger_refresh()
region: str → str
Returns an informational message that data refresh is not available via the bot (Playwright cannot run on Render free tier). Does not execute any scraping.
called when user asks to refresh or update data
check_subscription_status()
user_id: int → str
Queries MongoDB users collection for the user's subscription record. Returns subscribed regions and last notification date if subscribed.
called when user asks about their notification status
