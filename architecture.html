AGENT NODE
ChatMistralAI with 5 tools bound

Intent routing: detects regional / job-query patterns and injects a [MUST call …] hint into the current message — ephemeral, never persisted.

Conversation summarisation: if history > 20 messages, summarize_node fires first (using llm_plain, no tools) — condenses old msgs into one summary message, removes the rest via RemoveMessage.

System prompt: career assistant for Asia. Refuses off-topic. Telegram Markdown v1 format only.

FORMATTING GATEWAY
send_reply() is the only function that sends messages to users. Every handler, error path, and guardrail response passes through it.

Normalises LLM output to Telegram Markdown v1 (**bold** → *bold*, strips ### headers). On Telegram parse failure, strips all markdown symbols and retries as plain text — users never see raw *stars*.

db/models.py pymongo bulk_write
CONVERSATION STATE COLLECTIONS
checkpoints
Full AgentState (message history) per user, per graph step.
Stored as msgpack binary blobs — preserves Python object types faithfully.
Keyed by thread_id = str(telegram user_id).
Auto-created for every user who sends any message.
Survives bot restarts — conversation context is never lost.

checkpoint_writes
Write-ahead log. Partial states written here first before promotion to checkpoints. Enables crash recovery mid-graph-step.

Cleared per-user via /clear command.
MongoDBSaver langgraph-checkpoint-mongodb ≥ 0.3.1 msgpack binary
05  /  infrastructure & deployment
RENDER FREE TIER
Web service, 512 MB RAM, shared CPU.
Playwright/Chromium is not installed — pipeline runs from local machine only.

Health server: _ThreadingHTTPServer binds to $PORT on startup. Each ping is handled in its own thread, so the event loop is never blocked. Render and UptimeRobot can always reach /health.

Self-ping keep-alive: self_ping_loop() hits RENDER_EXTERNAL_URL/health every 4 minutes via Render's own reverse proxy — counts as inbound traffic and resets the 15-minute inactivity timer. The service never sleeps.
bot/health.py render.yaml ThreadingMixIn
EXTERNAL SERVICES
MongoDB Atlas (M0 free)
All persistent data. IP whitelist must include 0.0.0.0/0 or Render's outbound IP.

Mistral AI
mistral-small-latest used for both job summarisation (pipeline) and the ReAct agent (bot). Separate instances: llm (tools bound) and llm_plain (no tools — summarisation only).

UptimeRobot
External monitor pings /health every 5 minutes. Paired with the internal self-ping so Render never sleeps even when UptimeRobot backs off its check interval.

Langfuse (optional)
Cloud observability — set LANGFUSE_PUBLIC_KEY + SECRET_KEY to enable.
